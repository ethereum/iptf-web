This is an example draft post. It's in the `_drafts/` folder, so it won't be published to the live site.

To preview draft posts locally, run:

```bash
bundle exec jekyll serve --drafts
```

Draft posts in the `_drafts/` folder don't need a date in the filename or frontmatter.
